# 紫微斗数

> 简介

紫薇斗数是混合天文学、地理学、数学、统计学及合乎逻辑的伦理学，再结合普通的常识和长时间的体验而成一种学问。当初发明的人，是将天上南斗星群、北斗星群、紫薇垣群星及其他的杂星，来象征人的一生遭遇。就是将所生当年月日时，群星的方位躔度排列出来，研究它们相互间的感应。
